#  Pandas & Matplotlib Data Analysis

##  Overview

This repository contains hands-on practice notebooks demonstrating **data analysis and visualization** using **Pandas** and **Matplotlib** in Python.

It covers essential concepts such as data manipulation, handling missing values, and generating insightful visualizations.

---

##  Features

* Data analysis using **Pandas**
* Handling missing values (NaN)
* Working with **DataFrames** and **Series**
* Data visualization using **Matplotlib**
* Histograms and basic plots

---


##  Sample Visualizations

The notebook includes:

* Histogram of Math Scores
* Histogram of Science Scores

---

##  Technologies Used

* Python
* Pandas
* Matplotlib
* NumPy

##  How to Run

1. Clone the repository:

```bash
git clone https://github.com/Aaditya69Kumar/pandas-matplotlib-data-analysis.git
```

2. Navigate to the project folder:
```bash
cd pandas-matplotlib-data-analysis
```
3. Open the notebook:

```bash
jupyter notebook
```
